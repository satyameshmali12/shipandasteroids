
# Arrow And Asteroids

A 2d single player game

In this game,player has to hit the asteroid and thereby gain the points.

Player has three lives

## Controls

At a time player can either rotate or move.


```bash
  A - To rotate left
```

```bash
  D - To rotate right
```

```bash
  W - To move forward
```

```bash
  SPACE - To shoot bullet
```
